# ServeShare 🤝

*A Platform to Share Excess Food and Reduce Waste*

## 📌 Project Overview

**ServeShare** is a web-based application designed to connect food donors with people or organizations in need. The goal of this project is to **reduce food waste** and **support communities** by enabling easy and efficient food sharing.

Users can donate surplus food, view available donations, and request food based on their needs. The platform ensures transparency, accessibility, and ease of use for everyone involved.

---

## 🎯 Objectives

* Reduce food wastage
* Help needy individuals and organizations
* Create a simple and reliable food donation system
* Encourage social responsibility through technology

---

## 👥 Users of the System

* **Donors** – Individuals or organizations who want to donate surplus food
* **Receivers** – NGOs, shelters, or individuals who need food
* **Admin** – Manages users, donations, and system activities

---

## ✨ Features

* User registration and login
* Food donation posting
* View available food donations
* Request food donations
* Donation status tracking (Pending / Accepted / Completed)
* User profile management
* Admin dashboard for monitoring activities

## ⚙️ How to Run the Project

1. Clone the repository

   ```
   git clone https://github.com/your-username/serveshare.git
   ```

2. Install dependencies

   ```
   npm install
   ```

3. Set up the database

   * Import `serveshare.sql` into MySQL
   * Update database credentials in config file

4. Start the server

   ```
   npm start
   ```

5. Open browser and visit

   ```
   http://localhost:3000
   ```
